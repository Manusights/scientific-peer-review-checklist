# High Quality Review Example (Condensed)

## Summary
This manuscript tests whether intervention X improves outcome Y in cohort Z. The question is important, but the current analyses do not yet isolate causality.

## Major concerns
1. Confounding is not addressed in the primary model.
2. Key subgroup claim is underpowered.
3. Figure 3 does not match text claims.

## Minor concerns
- Define endpoint timing in Methods.
- Add confidence intervals in Table 2.

## Recommendation
Major revision. The paper can proceed if causality claims are reduced or supported with stronger analysis.
